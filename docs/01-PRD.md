# 01 - Product Requirements Document

## Nama Produk

Sadar Uang

## Deskripsi Singkat

Sadar Uang adalah web app pencatatan cash flow pribadi untuk mencatat pemasukan dan pengeluaran, lalu menampilkan ringkasan keuangan dalam dashboard harian, bulanan, dan tahunan.

## Masalah yang Diselesaikan

Banyak orang merasa uangnya cepat habis, tetapi tidak tahu uang tersebut habis untuk apa. Aplikasi ini membantu pengguna mencatat transaksi dengan mudah dan melihat pola pengeluaran secara visual.

## Tujuan Produk

1. Membantu pengguna mencatat uang masuk dan uang keluar.
2. Menampilkan dashboard keuangan yang mudah dipahami.
3. Membantu pengguna mengetahui kategori pengeluaran terbesar.
4. Membantu pengguna lebih sadar terhadap kebiasaan finansial pribadi.
5. Menjadi produk portofolio dan personal branding.

## Target Pengguna Tahap Awal

- Pengguna pribadi.
- Orang yang ingin mulai mencatat keuangan.
- Pengguna yang merasa spreadsheet terlalu ribet.
- Pengguna desktop dan mobile.

## Value Proposition

> Catat uang masuk dan keluar dengan simpel, lalu lihat ke mana uangmu pergi lewat dashboard yang mudah dipahami.

## Platform

- Web app responsif.
- Mendukung desktop dan mobile browser.
- Belum dibuat sebagai Android/iOS native app.

## Fitur Utama

### 1. Authentication

- Register
- Login
- Logout
- Proteksi halaman dashboard

### 2. Transaction Management

- Tambah transaksi
- Lihat transaksi
- Edit transaksi
- Hapus transaksi
- Filter transaksi
- Search transaksi

### 3. Dashboard

- Total pemasukan bulan ini
- Total pengeluaran bulan ini
- Sisa uang bulan ini
- Total transaksi bulan ini
- Grafik harian bulan berjalan
- Grafik bulanan dalam satu tahun
- Grafik pengeluaran berdasarkan kategori
- Riwayat transaksi terbaru

### 4. PDF Report

- Pilih periode bulan
- Generate laporan bulanan
- Download laporan PDF

## Non-Goals Versi Pertama

Fitur berikut tidak masuk MVP:

- Multi-dompet
- Budget bulanan
- Hutang/piutang
- Scan struk
- AI insight
- Reminder WhatsApp/Telegram
- Subscription/payment
- Multi-user keluarga
- Integrasi bank/e-wallet

## Success Metrics MVP

1. User bisa mencatat transaksi kurang dari 10 detik.
2. User bisa melihat ringkasan bulan berjalan.
3. User bisa mengetahui kategori pengeluaran terbesar.
4. User bisa export laporan PDF bulanan.
5. Aplikasi nyaman digunakan di desktop dan mobile.
6. User bisa memakai aplikasi selama 30 hari tanpa merasa ribet.
